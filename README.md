# 魔众微信管理系统

魔众微信管理系统是一套微信公众号管理系统，支持公众号菜单、自动回复等功能，有高扩展性。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
